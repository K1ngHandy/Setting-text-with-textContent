Getting Started

The following practice problems focus on using the Node.textContent property to make changes to the DOM

You are expected to understand how to target dom elements using the querySelector and getElementBy methods

Bonus points if you know the difference between Node.textContent and HTMLElement.innerText

You are encouraged to reference MDN as frequently as needed
